Guía rápida Flipper Zero - Herramientas y usos

¿Qué es el Flipper Zero?

Es un dispositivo multifunción de bolsillo diseñado para pruebas de seguridad en el mundo real. Permite interactuar con tarjetas RFID/NFC, señales de infrarrojos, sistemas de radiofrecuencia, dispositivos GPIO, BadUSB, iButtons y mucho más. Es ideal para aprender, investigar o testear sistemas.

Funciones principales

  • RFID 125kHz: Leer/emular tarjetas de baja frecuencia.
  • NFC 13.56MHz: Leer/clonar tarjetas MIFARE.
  • Infrarrojos: Capturar y reproducir señales de mandos.
  • Radiofrecuencia: Enviar/recibir señales de garajes, sensores, etc.
  • GPIO/UART/SPI/I2C: Conectar con dispositivos electrónicos.
  • iButton: Leer chips tipo llavero de contacto.
  • BadUSB: Ejecutar scripts como si fuera un teclado.

¿Qué necesitas?

  • Flipper Zero
  • Cable USB-C
  • Firmware oficial o custom (ej: Unleashed, RogueMaster)
  • PC o teléfono para gestionarlo
  • Tarjetas blancas si vas a clonar

Primeros pasos

  1. Instala QFlipper desde flipperzero.one.
  2. Conecta tu Flipper al PC o teléfono.
  3. Actualiza el firmware y explora las apps del dispositivo.

Ejemplos rápidos

  • Clonar una tarjeta RFID: Menú → RFID → 125kHz → Leer → Guardar → Emular.
  • Capturar un mando IR: IR → Capturar → Pulsa el mando → Guardar → Repetir.
  • Enviar señal RF: RF → Leer frecuencia → Capturar → Guardar → Transmitir.

Consejo final

Explora, prueba, pero siempre con responsabilidad y respeto por la ley. El Flipper es una herramienta educativa, no un juguete para molestar.
