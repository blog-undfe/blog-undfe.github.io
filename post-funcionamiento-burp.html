Funcionamiento básico de Burp Suite

Introducción:

Burp, o Burp Suite, es una herramienta gráfica para probar la seguridad de aplicaciones web. Está escrita en Java y desarrollada por PortSwigger Web Security. Su creador es la empresa Portswigger, alias de su fundador, Dafydd Stuttard.

Burp Suite es una plataforma integrada para realizar pruebas de seguridad de aplicaciones web. Sus diversas herramientas funcionan a la perfección para respaldar todo el proceso de pruebas, desde el mapeo inicial y el análisis de la superficie de ataque de una aplicación hasta la detección y explotación de vulnerabilidades de seguridad.

Burpsuite puede utilizarse como un proxy HTTP básico para interceptar tráfico para su análisis y reproducción, como escáner de seguridad de aplicaciones web, como herramienta para realizar ataques automatizados contra aplicaciones web, como herramienta para rastrear sitios web completos e identificar la superficie de ataque y cuenta con una API de plugins con una gran cantidad de complementos de terceros disponibles.

Es la herramienta más popular entre los investigadores profesionales de seguridad de aplicaciones web y los buscadores de errores. Su facilidad de uso la convierte en una opción más adecuada que alternativas gratuitas como OWASP ZAP.

Para utilizar Burp Suite correctamente, es necesario conocer todas las herramientas disponibles. En el próximo artículo, utilizaremos la versión Burp Community, ya que está disponible para todos de forma gratuita.

Diferentes ediciones de Burp Suite:

  • Community: Versión gratuita que se instala por defecto en Kali Linux.
  • Profesional: Edición de pago, cuesta alrededor de $399 al año.
  • Enterprise: Edición para empresas, cuesta alrededor de $3999 al año.

Características:

  • Enterprise ($3999/año): Escáner de vulnerabilidades web, escaneos programados y repetidos, escalabilidad ilimitada, integración de CI.
  • Profesional ($399/año): Escáner web, herramientas manuales avanzadas y esenciales.
  • Community (Gratis): Herramientas manuales esenciales.
